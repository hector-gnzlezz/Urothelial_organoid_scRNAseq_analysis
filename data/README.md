# Data

Raw data are publicly available on NCBI GEO and are not included 
in this repository due to file size.

## Source

**GEO Accession: [GSE131909](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE131909)**

Single cell RNA-seq of Proliferative (NMU-O-P) and Differentiated (NMU-O-D) 
Normal Mouse Urothelium Organoids.

- Organism: *Mus musculus*
- Platform: Illumina NextSeq 500
- Samples: 6,826 cells (NMU-O-P) and 4,896 cells (NMU-O-D)
- Publication: [PMID 31562298](https://pubmed.ncbi.nlm.nih.gov/31562298/)
- Contributors: Pereira Santos C, Martinez Villarreal J, Real F (CNIO, Madrid)

## How to download

### Option 1: Direct download
```bash
wget https://www.ncbi.nlm.nih.gov/geo/download/?acc=GSE131909&format=file \
     -O GSE131909_RAW.tar
tar -xvf GSE131909_RAW.tar
```

### Option 2: R (GEOquery)
```r
install.packages("BiocManager")
BiocManager::install("GEOquery")

library(GEOquery)
getGEOSuppFiles("GSE131909", baseDir = "./data/raw/")
```

### Additional files (tracked in this repository)

- `HK_genes_mouse.txt` — Curated mouse housekeeping gene list used as an 
  additional QC metric. Cells expressing fewer than 40 housekeeping genes 
  are flagged as low quality and removed.

## Processed data (`processed/`)

Generated automatically by running the pipeline scripts in order.
Not tracked by git.

| File | Generated by | Description |
|------|-------------|-------------|
| `NMU_O_integrated_markers.txt` | script 04 | Differential markers per cell population |
| `NMU_O_gene_sets.gmt` | script 04 | Cell population signatures for ssGSEA |
| `NMU_O_expr_matrix.tsv` | script 04 | Reference matrix for CIBERSORTx deconvolution |
| `scRNAseq.RData` | script 04 | Full R session checkpoint |
