# Results

Output files generated by the analysis pipeline.
Not tracked by git — regenerate by running the scripts in order.

## Contents

### `figures/`
Plots generated at each step of the pipeline.
See `figures/README.md` for a full list.

### Marker tables
| File | Description |
|------|-------------|
| `NMU_O_integrated_markers.txt` | Differential markers for all annotated cell populations in the integrated dataset. Columns: gene, p_val, avg_log2FC, pct.1, pct.2, p_val_adj, cluster |

### Gene signatures
| File | Description |
|------|-------------|
| `NMU_O_gene_sets.gmt` | Cell population gene signatures in GMT format, ready for ssGSEA via GenePattern |

### Deconvolution reference
| File | Description |
|------|-------------|
| `NMU_O_expr_matrix.tsv` | Single-cell reference matrix for CIBERSORTx deconvolution of bulk RNA-seq data. 10 cells per population, gene symbols converted to human orthologs |

### Session
| File | Description |
|------|-------------|
| `scRNAseq.RData` | Full R session saved after script 04. Load with `load("results/scRNAseq.RData")` to resume analysis without re-running the pipeline |
